What I do

iOS Full Stack

My current learning obsession is mobile iOS app development in iOS 13 with Swift 5. I intend to be procifient enough to build some of the project apps I have in mind in 3 months or less, pending work at JPMorgan Chase & Co.

My first foray into mobile development was with the open source cross platform mobile framework being developed by Google, "Flutter". Since then, I've realized that the proper path to mobile mastery was excellence in the native space first. I chose iOS because of the syntax of the Swift language and the excellent mobile ecosystem that Apple has developed. I'm currently working through Angela Yu's excellent Udemy course on iOS engineering to round out my proficiency.

Python Backend Development

Python is by far my strongest language, and for good reason. It's expressiveness and concise syntax makes it my go to for backend tooling that pulls several systems together into unified functionality. Python is my first choice language for quick algorithm implementations during software engineering interviews, and of course it is the de facto Machine Learning language (another of my interests)

At work I mostly use Python to develop orchestration and automation tools for client software engineers in the firm and to make life easier for the engineers on my team.

Full Stack Web Development

Along with this site, I have been building small scale web apps since college (December 2019) using HTML/CSS/JS using Node as the backend with Express and SQL for server and database management, respectively. I see web development as an essential tool for any software engineer and one that must be constantly honed.

Currently going through Angela Yu's excellent Udemy course on Web Development to round out my knowledge. At work I dabble in the Spring Java Framework for the back end and Angular for the front end developing automation apps for client software engineers at JPMorgan Chase & Co.

Machine Learning

I have been studying and working with Machine Learning model pipelines with Python since senior year of college, and it is an area of software engineering that fascinates me greatly.

I have some slight work experience in ML from a team I was briefly on that worked with natural language processing pipelines for automation engineering. Currently working through the Udemy course "Complete Machine Learning and Data Science: Zero to Mastery" by Andrei Neagoie and Daniel Burke to supplement my knowledge.

Who I am
